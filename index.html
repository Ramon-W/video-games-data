About:

This webapp contains video game data from The CORGIS Data Set Project, containing 
        1212 titles from the year 2004 to 2009. It includes the general information about each game like publisher, sales, and review scores.
        
        One set of statistics for each video game is the playtime, crowdsourced from HowLongToBeat.com. This includes average 
        time of all playstyles, as well as individual playstyles. If a playstyle for a game does not exist (e.g. completionists), then it is denoted as 0 hours. 
        Additionally, if the game does not have a specific end or there is no sufficient data, every time would be marked as 0 hours. 
        
        Publisher may be left blank if the publisher of the video game is unknown. Ratings are from ESRB,
        but only includes E, T, and M. The review score is out of 100, and from Metacritic.
